# 🌾 Farmer Assistant Chatbot

A multilingual chatbot to assist farmers with crop advice,
weather updates, market prices, and pest control guidance.

## Languages Supported
- English
- Hindi (हिंदी)
- Telugu (తెలుగు)

## Features
- Real-time weather updates
- Crop advisory by season
- Market price information
- Pest & disease guidance
- Multilingual interface

## Tech Stack
- Frontend: HTML, CSS, Vanilla JavaScript
- Translations: Custom JSON-based i18n
- Weather: OpenWeatherMap API
- Hosting: GitHub Pages

## How to Run
1. Download or clone this repo
2. Open index.html in your browser
3. No build step needed!
